Imaging Handler

   Imaging Handler is probably the most complex piece that comes with Najm. It is responsible for loading FITS image data for an object, creating a color scale and color map, then use them to create and display an image of the object.

Color Scaling:

   Image Handler allows you to choose your color scale from among several supported ones, to even create your own scaling algorithm in python (still under construction!), and to see a color histogram and change min/max limits.

Color Mapping:

Najm Image Handler support 18 different built in color maps, custom color maps loaded from text files, and a unique advanced color mapping technique allow you to start from a built in color map and customize it using linear interpolation with few mouse clicks. You can also control the contrast and bias of the result color map.
